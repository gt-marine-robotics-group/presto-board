# PRESTO Board

The PRESTO (Peripheral, Radio, and E-Stop) Mainboard project is intended to provide interfaces for standard servo-based peripherals and the emergency stop. Functions that PRESTO provides include direct servo commands, receiving radio control signals, triggering the emergency stop, sensing the emergency stop, and outputing status to a light tower.

## Board Variants

### Presto
Presto is the full functionality variant, designed with as many servo interfaces as possible.

### Presto Jr.
Presto Jr. is a reduced functionality variant, with fewer servo interfaces.
